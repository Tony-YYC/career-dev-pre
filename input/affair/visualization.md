## 三、可视化和生成图片建议

（仅供参考，可以按实际情况调整）

最值得可视化的不是“把文字换成图标”，而是把这部分讲成一条证据链。建议重点做这几类图：

1. **总逻辑路线图**
   放在Slide 1或2：科学创始人 → Generate Platform → Chroma论文 → 专利 → 临床管线 → 合作验证。这样一开始就让听众知道你不是堆资料，而是在讲一个完整逻辑。

2. **传统药物发现 vs Generate药物生成对比图**
   左边画“自然分子库/抗体库 → 筛选 → 多轮优化 → 候选药物”，右边画“目标条件 → AI生成 → 高通量验证 → 模型学习 → 候选药物”。这页会比纯文字解释优势更清楚。

3. **Design–Build–Test–Learn闭环图**
   这是核心平台页最重要的图。可以自己重画，不要直接截图招股书小图。建议用四个圆环节点，旁边分别放“AI模型、DNA组装、蛋白表达/检测、数据回流”。

4. **Chroma机制简化图**
   你们Word里已有Chroma论文图，但原图信息量大。PPT里可以保留原图一角作为“论文出处”，主图则重画为：噪声蛋白结构 → 扩散去噪 → 条件约束 → 生成蛋白结构/序列。

5. **Chroma生成蛋白展示墙**
   Word里已有字母形状蛋白图，可以放在Slide 7或8作为视觉亮点。旁边配一句话：“模型不是只预测结构，而是可以按形状、对称性、功能条件生成结构。”

6. **GB-0895作用机制图**
   这张建议生成或重画：上皮细胞释放TSLP → 激活炎症通路 → anti-TSLP抗体阻断 → 减少炎症。用简洁医学插画风，比文字说“TSLP是上游因子”更直观。

7. **SOLAIRIA临床试验流程图**
   Slide 12建议做成时间轴：入组 → 随机分组 → Week 0给药 → Week 26给药 → Week 52主要终点。最好标出“300mg SC Q26W”和“annualized asthma exacerbation rate”。

8. **专利护城河三层图**
   平台层：Chroma/扩散模型；应用层：交叉反应性抗体、功能优化；产品层：GB-0895等具体分子。比列专利表更适合演示。

9. **合作生态图**
   Generate放中心，四周放Amgen、Novartis、MD Anderson、Roswell Park。每条线写一句合作价值：大药企验证、资金、靶点资源、临床转化资源。

10. **结尾天平图**
    左边是“技术证据”：论文、专利、临床管线、合作；右边是“未验证风险”：Phase 3结果、AI监管、制造放大、IP不确定性。结尾会更有判断力。

建议PPT中文字控制在每页3—5个短句以内，真正要讲的内容放进speaker notes。这样“事”的部分既能体现资料扎实，也不会变成满屏文字。
